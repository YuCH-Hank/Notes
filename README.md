# Pipe Selector (Web 版)

Pipe Selector 是一個將原本 Python / Tkinter 管徑計算工具，完整改寫成 **純前端 Web App** 的專案。  
可以直接部署於 **GitHub Pages**，在 iOS / Android / 電腦上使用，也能加入主畫面當作「類 App」。

---

## 📌 功能總覽（管路系列工具集）

這個小工具 App 目前包含 3 個模組（可在頁面上用分頁切換）：

### 1. 風量計算器（Flow Calculator）

- 輸入多組風量與風速（支援多種單位）
- 自動計算：
  - 每段風量
  - 累積風量
  - 需求截面積（m²）
- 根據標準管徑表推薦圓管尺寸
- 以原生 HTML `<table>` 呈現結果，方便複製或截圖

---

### 2. 四角轉圓（Rect → Round）

- 輸入矩形管道的長、寬
- 支援長寬單位：公尺 / 公分 / 毫米
- 自動計算矩形面積（m²）
- 推薦等效圓管管徑（mm），同時顯示等效圓面積（m²）

---

### 3. 風量快查表（Quick Lookup）

- 輸入：
  - 固定風速（例如 10、12、13 m/s）
  - 風量範圍（起始、結束、步進）
- 自動產生一張快查表，內容包含：
  - 風量（CMM）
  - 風量（CMS）
  - 需求截面積（m²）
  - 理論圓管直徑（mm）
  - 推薦標準管徑（mm）

這個模組就是把工程常用的「風量 ↔ 管徑快查表」變成互動化版本。

---

## 🌓 深色模式 & 手機 App 風格

- 支援 **深色 / 淺色模式切換**（右上角 🌙 / ☀️ 按鈕）
- 會記住使用者選擇（使用 `localStorage`）
- 手機上採用：
  - 上方固定標頭（App 樣式 Title）
  - 三個工具以「分頁 Tab」切換，只顯示一個畫面
  - 卡片式布局，適合直立使用

---

## 📁 專案架構

```text
my-web-app/
├─ index.html         # HTML 主頁：App 框架 + 三個工具區塊
├─ style.css          # CSS：深/淺色主題、排版、表格、App 樣式
└─ src/
   ├─ defines.js      # 常數與參數：單位換算、管徑列表、面積表
   ├─ calculator.js   # 核心計算邏輯：面積、累積和、推薦管徑
   ├─ ui.js           # UI 邏輯：分頁切換、深色模式、計算綁定
   └─ main.js         # 程式入口：DOM 載入後啟動 UI
```
